This inspection checks if namespace prefix is derived from file name part in namespace URI. Inspection expects that URI is based on real file name. File extension should not be part of namespace prefix. For example in module with namespace URI my://sample/namespace/myFirstModule.xq namespace prefix should be myModuleOne. 
module namespace myFirstModule = "my://sample/namespace/myFirstModule.xq";

declare function myFirstModule:someFunction() {
  fn:true()
};
