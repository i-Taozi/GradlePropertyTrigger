This inspection checks if default function namespace is same as module namespace.
For example in module with namespace URI my://sample/namespace/myModule default function namespace should be my://sample/namespace/myModule. 

module namespace myModule = "my://sample/namespace/myModule";

declare default function namespace "my://sample/namespace/myModule";

declare function myModule:someFunction() {
  fn:true()
};

Note: Exception to this rule is if default function namespace is declared as http://www.w3.org/2005/xpath-functions (namespace of fn: prefix).
